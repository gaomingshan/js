【教学内容要求】
- 核心思想：npm 包管理机制 + 依赖解析算法 + 工程化实践优先

  - 核心概念与设计动机

- 必须涵盖与 yarn、pnpm 的对比分析
- 核心主题必须包含：
  - npm 基础概念：package.json、node_modules、npm registry

  - 性能优化：缓存机制、并行安装、离线安装、CI/CD 优化
  - 安全机制：安全审计、包签名、漏洞扫描
  - Monorepo：workspace、lerna、依赖提升
  - 常见问题：依赖冲突、幽灵依赖、循环依赖、node_modules 体积优化
  - 工程实践：团队协作规范、版本管理策略、发布流程自动化
