你是一名资深前端架构师与构建工具专家，在当前目录下，请生成 Vite 系统化学习内容，要求如下：

【学习对象】
- 有前端工程化经验，熟悉 Webpack 等传统构建工具，希望掌握 Vite
- 使用过 Vite 脚手架，但对其核心原理、配置优化、插件机制理解碎片化
- 希望深入理解 Vite 设计思想、工作原理及工程实践最佳方案

【生成顺序与文件结构】
1. 生成大纲文件 content-outline.md，每个小节带有 Markdown 跳转链接到对应 content 文件
2. 生成教学内容，存放于 content/ 目录
3. 生成实战案例，存放于 examples/ 目录
4. 生成面试题汇总，存放于 quiz/quiz.md

【文件命名规则】
- 大纲文件：content-outline.md
- 教学内容：content/content-1.md、content/content-2.md … 按大纲顺序递增
- 实战案例：examples/example-1.md、examples/example-2.md … 按场景分类
- 面试题汇总：quiz/quiz.md
- 链接格式：[小节标题](./content/content-x.md)，x 按大纲顺序递增
- 统一使用 Markdown，中文，技术严谨且教学友好

【教学内容要求】
- 核心思想：Vite 设计理念 + 开发服务器原理 + 生产构建优化优先
- 每小节包含：
  - 核心概念与设计动机
  - 工作原理与底层机制
  - 配置方式与最佳实践
  - 常见问题与解决方案
  - 实际工程应用示例
- 精简但完整，避免简单罗列配置项
- 难度分级：快速上手 → 深入配置 → 原理剖析 → 高级应用
- 必须对比 Vite 与 Webpack 的差异，帮助理解设计优势
- 教学内容和示例要覆盖真实项目场景
- 核心主题必须包含：
  - Vite 设计理念与核心特性（ESM、esbuild、Rollup）
  - 开发服务器原理（即时模块热更新、预构建、依赖优化）
  - 生产构建流程（代码分割、资源优化、浏览器兼容）
  - 配置系统详解（开发配置、构建配置、优化配置）
  - 插件机制与生态（插件 API、钩子系统、常用插件）
  - 静态资源处理（CSS、图片、字体、JSON 等）
  - 环境变量与模式配置
  - TypeScript 与 JSX 支持
  - 多页面应用与库模式
  - 性能优化实践（冷启动、热更新、构建速度、产物体积）
  - 迁移指南（从 Webpack/Vue CLI 迁移到 Vite）
  - SSR 与 SSG 支持
  - 底层原理深入（模块解析、依赖图、HMR 实现、预构建机制）

【实战案例要求】
- 场景覆盖：Vue3/React/Svelte 项目、组件库、Monorepo、SSR 等
- 每个案例包含：
  - 场景描述与需求分析
  - 完整配置文件示例
  - 关键配置解释
  - 常见坑点与解决方案
- 所有配置代码必须可直接使用
- 案例按实际项目难度递进

【面试题要求】
- 题型：单选 / 多选 / 代码题 / 场景题结合，20-30 题
- 难度递进：基础概念 → 配置优化 → 原理剖析 → 综合应用
- 题目覆盖大纲核心内容，偏向实际应用与问题解决
- 每题提供答案与解析，解析分层：
  - 答案要点
  - 原理解释
  - 实践建议
  - 易错点提示
- 面试题仅生成一个汇总文件 quiz/quiz.md
- 面试题解析保持简洁，突出核心要点

【大纲设计原则】
- 学习路径清晰：快速入门 → 核心功能 → 配置详解 → 原理剖析 → 高级应用 → 工程实践
- 逻辑递进，章节之间有依赖关系
- 每章节标题明确、简洁，并说明本章解决的核心问题
- 避免简单罗列 API 或配置项，强调理解与应用
- 保证 Vite 核心概念（开发服务器、预构建、HMR、插件系统、生产构建）覆盖完整

【质量红线】
- 禁止空洞、堆砌、简单翻译官方文档
- 小节概念不清晰时应缩减而非堆砌
- 所有示例配置必须可执行并贴近真实项目场景
- 所有原理解析必须准确，不允许模糊或猜测
- 配置示例必须包含注释说明，便于理解
- 性能优化建议必须有明确的量化指标或对比
- 不允许混淆开发模式与生产模式的配置

【最终目标】
- 一条 Prompt 生成：
  1. content-outline.md（大纲）
  2. content/ 目录下的章节内容（教学 + 配置示例）
  3. examples/ 目录下的实战案例（完整配置方案）
  4. quiz/quiz.md（面试题）
- 确保学习者：
  - 理解 Vite 设计理念与核心优势
  - 掌握完整的配置能力与优化技巧
  - 能够独立搭建和优化 Vite 项目
  - 理解底层原理，具备问题排查能力
  - 能够根据项目需求定制 Vite 方案
  - 能顺利应对 Vite 相关面试题
