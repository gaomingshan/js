性能优化指南

一、代码层面

  • 避免不必要的循环
  • 使用高效的数据结构（Map/Set）
  • 减少 DOM 操作
  • 使用事件委托
  • 防抖和节流

二、加载优化

  • 代码分割（Code Splitting）
  • 懒加载（Lazy Loading）
  • Tree Shaking
  • 压缩和混淆
  • 使用 CDN

三、运行时优化

  • 避免强制同步布局
  • 使用 requestAnimationFrame
  • Web Workers 处理密集计算
  • 优化图片加载

💻 交互式代码示例

示例：防抖和节流

console.log('=== 防抖 ===\n');

function debounce(fn, delay) {
    let timer;
    return function(...args) {
        clearTimeout(timer);
        timer = setTimeout(() => fn(...args), delay);
    };
}

const search = debounce((query) => {
    console.log('搜索:', query);
}, 300);

search('a');
search('ab');
search('abc');  // 只会执行这一次

setTimeout(() => {
    console.log('\n节流示例（概念）');
}, 500);
