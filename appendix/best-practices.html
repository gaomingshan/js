最佳实践与代码规范

一、命名规范

  • 变量和函数：camelCase
  • 类和构造函数：PascalCase
  • 常量：UPPER_SNAKE_CASE
  • 私有字段：#privateField

二、代码组织

  • 一个文件一个类/模块
  • 相关功能分组
  • 使用 ES6 模块
  • 避免全局变量

三、编码习惯

  • 使用 const/let，避免 var
  • 优先使用箭头函数
  • 使用模板字符串
  • 使用解构赋值
  • 使用可选链和空值合并

四、性能优化

  • 避免不必要的重复计算
  • 使用 debounce/throttle
  • 合理使用缓存
  • 避免内存泄漏

💻 交互式代码示例

示例：最佳实践

console.log('=== 最佳实践 ===\n');

// ✓ 好的写法
const getUserInfo = (user) => {
    const { name, age, email } = user;
    return `${name}, ${age}, ${email}`;
};

const user = {
    name: 'Alice',
    age: 25,
    email: 'alice@example.com'
};

console.log(getUserInfo(user));

// 使用可选链
const city = user?.address?.city ?? 'Unknown';
console.log('城市:', city);
