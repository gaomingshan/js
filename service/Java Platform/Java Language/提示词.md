# 【教学内容要求】

每小节必须包含：

- 核心概念  
- 设计动机与语言哲学  
- 编译期行为  
- 字节码与 JVM 执行机制  
- 易错点/边界行为  
- 实际场景或推导示例（无 API 教程）

核心主题必须覆盖：

- Java 语言设计哲学（单继承、Object 根类、接口常量设计、final 语义）  
- 类型系统与特殊语法（泛型擦除、桥方法、三目运算类型提升、自动装箱/拆箱）  
- 特殊语法原理（equals/hashCode 设计契约、finally 执行机制、枚举类原理、注解底层机制）  
- 编译期机制与字节码（语法糖展开、lambda/invokedynamic、内部类捕获外部变量）  
- JVM 执行模型（内存分区、对象布局、堆/栈/方法区、程序计数器）  
- 垃圾回收机制（分代收集、新生代/老年代、标记-清除、标记-整理、复制算法、Stop-The-World、CMS/G1 思想）  
- 类加载机制（五阶段加载、双亲委派、静态变量初始化顺序、常量池）  
- Java 内存模型（happens-before、volatile、synchronized、final 字段语义）  
- JIT 与运行期优化（解释执行 vs JIT、内联、逃逸分析、栈上分配、锁优化）  
- 语言边界行为（编译期常量折叠、异常执行顺序、语法特殊行为推导）
