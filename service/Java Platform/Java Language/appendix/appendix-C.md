# 附录C：学习资源推荐

## 书籍推荐

### 核心书籍

1. **《深入理解 Java 虚拟机（第3版）》** - 周志明
   - JVM 原理最佳中文书籍
   - 涵盖内存模型、GC、类加载、JIT

2. **《Effective Java（第3版）》** - Joshua Bloch
   - Java 编程最佳实践
   - Google Java 首席架构师经验总结

3. **《Java 并发编程实战》** - Brian Goetz
   - 并发编程权威指南
   - Java 并发包作者亲著

4. **《Java 性能权威指南》** - Scott Oaks
   - 性能调优全面指南
   - 涵盖 JVM、GC、监控

### 进阶书籍

5. **《Java 语言规范》**（JLS）
   - 官方语言规范
   - 语言细节权威参考

6. **《Java 虚拟机规范》**（JVMS）
   - 官方 JVM 规范
   - 字节码和执行模型

## 工具推荐

### 开发工具

- **IntelliJ IDEA**：最强 Java IDE
- **Eclipse**：开源 IDE
- **Maven/Gradle**：构建工具

### 性能分析

- **VisualVM**：JVM 监控和分析
- **JProfiler**：商业性能分析工具
- **async-profiler**：低开销 CPU/内存分析
- **Arthas**：阿里开源诊断工具

### GC 分析

- **GCEasy**：在线 GC 日志分析
- **GCViewer**：GC 日志可视化
- **MAT（Memory Analyzer Tool）**：堆转储分析

## 实践项目

### 推荐练手项目

1. **实现简单的 JVM**
   - 理解字节码执行
   - GitHub: mini-jvm

2. **实现 HashMap**
   - 理解哈希表原理
   - 实现扩容、红黑树

3. **实现线程池**
   - 理解并发编程
   - 实现任务队列、线程管理

4. **实现对象池**
   - 理解对象复用
   - Apache Commons Pool

## 学习路径建议

### 初学者（0-6个月）

1. Java 基础语法
2. 面向对象编程
3. 常用 API 和集合框架
4. 异常处理和 I/O

### 进阶（6-12个月）

1. 多线程和并发
2. JVM 内存模型
3. 垃圾回收机制
4. 设计模式

### 高级（12个月+）

1. JVM 调优
2. 高并发编程
3. 性能优化
4. 源码阅读（JDK、Spring）
