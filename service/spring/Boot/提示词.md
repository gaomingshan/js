# 【教学内容要求】

每小节必须包含：

- 核心概念与设计思想  
- 技术架构与类图结构  
- 核心类/接口/方法详解  
- 源码实现原理与执行流程  
- 生命周期与扩展点机制  
- 实际落地场景与最佳实践  
- 常见问题与易错点

核心主题必须覆盖：

- **Spring Boot 核心特性**（自动配置、starter 依赖、嵌入式容器、生产就绪 Actuator、条件注解）  
- **自动配置原理**（@SpringBootApplication 注解、@EnableAutoConfiguration、AutoConfigurationImportSelector、spring.factories 加载机制）  
- **条件装配机制**（@Conditional 系列注解、@ConditionalOnClass、@ConditionalOnBean、@ConditionalOnProperty、条件评估流程）  
- **Starter 机制深入**（Starter 设计原则、自定义 Starter 开发、依赖管理、自动配置类编写）  
- **配置管理体系**（application.properties/yml、配置加载优先级、@ConfigurationProperties、外部化配置、配置加密）  
- **嵌入式容器原理**（Tomcat/Jetty/Undertow 嵌入式启动、ServletWebServerFactory、容器定制化、多容器切换）  
- **生产级特性 Actuator**（端点配置、健康检查、指标监控、自定义端点、安全配置）  
- **Spring Boot 启动流程**（SpringApplication.run() 执行流程、ApplicationContext 初始化、环境准备、监听器机制）  
- **日志体系集成**（Logback/Log4j2 自动配置、日志级别控制、日志文件配置、自定义日志输出）  
- **数据源自动配置**（HikariCP 连接池、多数据源配置、JPA/MyBatis 集成、事务管理）  
- **缓存抽象与集成**（@EnableCaching、CacheManager、Redis/Caffeine 集成、缓存注解使用）  
- **安全框架集成**（Spring Security 自动配置、认证授权、JWT 集成、OAuth2 支持）  
- **消息队列集成**（RabbitMQ/Kafka 自动配置、消息监听、消息发送、事务消息）  
- **异步任务与定时任务**（@EnableAsync、@Async 注解、线程池配置、@EnableScheduling、@Scheduled 使用）  
- **测试支持**（@SpringBootTest、切片测试、MockMvc、TestContainers 集成）

**工作实战场景必须覆盖：**

- **启动性能优化**（懒加载配置、条件装配优化、排除不需要的自动配置、打包优化）  
- **配置管理实践**（多环境配置、配置加密、配置中心集成、配置热更新）  
- **故障排查方法**（自动配置失败排查、端点不可用分析、内存溢出定位、性能瓶颈分析）  
- **常见问题解决**（自动配置不生效、配置属性不生效、Actuator 端点问题、嵌入式容器问题）  
- **监控与运维**（Actuator 监控实践、自定义健康检查、指标采集、日志聚合、分布式追踪）  
- **微服务集成**（Spring Cloud 集成、服务注册与发现、配置中心、服务网关、熔断限流）  
- **容器化部署**（Docker 镜像构建、分层构建优化、Kubernetes 部署、健康检查配置）  
- **安全加固**（Actuator 端点安全、敏感信息过滤、HTTPS 配置、跨域处理）

**面试高频考点必须覆盖：**

- **对比类问题**（Spring Boot vs Spring、starter vs 传统依赖、jar 包部署 vs war 包部署、Tomcat vs Jetty vs Undertow）  
- **原理类问题**（自动配置原理、@SpringBootApplication 注解作用、条件装配如何工作、spring.factories 加载机制）  
- **场景类问题**（自动配置不生效排查、如何自定义 Starter、如何优化启动速度、如何实现配置加密）  
- **设计类问题**（为什么 Spring Boot 能简化开发、自动配置的设计思想、约定大于配置的实现、如何设计一个可扩展的 Starter）  
- **实战类问题**（如何实现多数据源、如何集成分布式配置中心、如何实现优雅停机、如何监控应用健康状态）

# 【面试题要求】

**面试题分级体系（约 150 题）：**

1. **初级岗位题库（30 题）** - 适合 0-2 年经验
   - Spring Boot 基本概念、核心注解、starter 使用
   - application.yml/properties 配置、常用配置项
   - Actuator 基本使用、内嵌容器配置
   
2. **中级岗位题库（50 题）** - 适合 2-5 年经验
   - 自动配置原理、@Conditional 条件装配
   - Starter 机制、配置加载优先级
   - 嵌入式容器原理、启动流程详解
   - 常见问题排查、源码关键流程理解
   
3. **高级岗位题库（40 题）** - 适合 5+ 年经验
   - 自动配置源码架构、SPI 机制深入
   - 启动性能优化、内存优化、故障定位
   - 自定义 Starter 开发、条件装配高级应用
   - 微服务架构设计、监控体系搭建
   
4. **架构岗位题库（30 题）** - 适合架构师/技术专家
   - Spring Boot 设计理念与约定大于配置
   - 微服务架构下的 Spring Boot 应用设计
   - 技术选型、性能调优、高可用设计
   - 团队开发规范与 Starter 标准化

**每题必须包含：**
- 题目（模拟真实面试场景）
- 标准答案（分点清晰）
- 原理深挖（含类图/时序图/源码片段）
- 追问方向（面试官可能的追问点）
- 加分项（超出预期的回答内容）
- 实战关联（工作中如何应用）
- 常见错误回答（避免踩坑）

**题型分布：**
- 概念辨析题：20%（考察理解深度）
- 原理解析题：30%（考察底层机制）
- 场景分析题：25%（考察实战能力）
- 源码追踪题：15%（考察源码阅读）
- 方案设计题：10%（考察架构思维）
