## 🎯 设计原则

1. **从底层到应用**：先理解CSS如何工作，再学怎么用
2. **重视原理**：第二部分占比最大（15章150题），深入CSS规范和浏览器实现
3. **深入但不脱离**：涉及渲染引擎，但始终围绕CSS本身
4. **循序渐进**：每部分都建立在前面的基础上
5. **数据与渲染分离**：JSON数据 + 渲染引擎，加速内容生成
